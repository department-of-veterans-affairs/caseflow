# Icons

Each Icon is its own component and takes at least 3 props - size, color, and className. The size prop sets the height of the component. The width is set automatically within the SVG element found in each component. 
The color prop sets the fill color within the SVG element. The className prop sets the className found on the SVG element in the component. 

Some icons will have more than 3 props. Some icons have no props. The ones with no props are Font Awesome icons that are used enough to deserve their own component.

The default size (height) for each icon is 20px.

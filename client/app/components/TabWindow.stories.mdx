# TabWindow

Displays content in sections based on the currently selected tab. Allows for grouping sections of a page based on common functionality
The US Web Design doesn’t include tabs so we’ve designed our own.
The active tab is highlighted with a blue border and blue text.
Inactive tabs have grey text. Disabled tabs are grey, have faded grey text,
and cannot be clicked.

### Default

The tab window will by default toggle between the tab list that is provided as props and render the page value of the active tab

<Preview>
  <Story name="Default">
    {() => {
      return <TabWindow tabs={defaultProps.tabs} />;
    }}
  </Story>
</Preview>

### Disabled

Tabs can be disabled by setting the `disable` prop to `true` on the tab you wish to disable

<Preview>
  <Story name="Disabled">
    {() => {
      return (
        <TabWindow
          tabs={[
            {
              disable: false,
              label: "Enabled Tab",
              page: "Accessible content",
            },
            {
              disable: true,
              label: "Disabled Tab",
              page: "Inaccessible content",
            },
          ]}
        />
      );
    }}
  </Story>
</Preview>

### Icons

Icons are optional on tabs and are set by the designer in the mockup.
These icons come from the Font Awesome package, which is set in the
U.S. Design standards. Here is an example of tabs with icons:

<Preview>
  <Story name="With Icons">
    {() => {
      return (
        <TabWindow
          tabs={[
            {
              icon: checkSymbolHtml(),
              disable: false,
              label: "Check Tab",
              page: "Content with icon",
            },
            {
              icon: crossSymbolHtml(),
              disable: false,
              label: "Cross Tab",
              page: "More content with icon",
            },
          ]}
        />
      );
    }}
  </Story>
</Preview>

### FullPage

You can also make the tab content span the full page by setting the `fullPage` prop to `true`.

            {

              label: "Full Page Tab 2",
              page: "More spanned content",
            },
          ]}
        />
      );
    }}
  </Story>
</Preview>
